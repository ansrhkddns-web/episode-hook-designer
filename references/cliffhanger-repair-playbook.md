# Cliffhanger Repair Playbook

Use this reference when an episode ending feels weak, vague, slow, or uncommercial. Repair by changing the last beat into a concrete unresolved pressure.

## Repair Formula

```text
current ending
-> identify missing pressure
-> move or create concrete object/action
-> tie it to protagonist action
-> stop before response
-> write one next-click question
```

## Common Weak Endings

| Weak Ending | Why It Fails | Repair |
| --- | --- | --- |
| "Something felt ominous." | no object, no action, no question | show the message, witness, door, rule, or person causing the ominous feeling |
| The protagonist goes to sleep | tension cools down | end before sleep, when the unresolved message/visitor/result arrives |
| Travel begins | transition is not pressure | make the travel interrupted by pursuit, summons, accident, or clue |
| Preparation starts | no immediate consequence | end on the discovered condition that changes the preparation |
| Explanation ends | reader has information but no pressure | turn the explanation into a test, accusation, or forced decision |
| Reward is fully received | no next desire | reveal price, watcher, hidden condition, or new rival |
| Defeat happens and stops | reader buys frustration | add information gain, survival choice, or means of counterattack |
| Conversation fades out | no line forces response | end on the sharpest question, confession, accusation, or refusal |
| New character appears | character has no meaning yet | attach name, power, relationship, threat, or desired answer |
| Vague mystery appears | reader cannot theorize | provide one concrete clue and one missing piece |

## Ending Object Bank

Use one concrete object or event:

- message
- name
- ranking
- contract

- witness

## Repair Types

### Move Later Beat Earlier

If the next episode starts with the real hook, move that first scene to the current ending and stop before the response.

### Add Price to Reward

If the episode ends on success, add a condition, watcher, debt, rival claim, hidden clause, or consequence.

### Add Witness to Private Action

If the protagonist acts alone, make someone meaningful see it.

### Add Forced Answer

If characters talk without consequence, end on a line that requires immediate answer:

- "Then prove it now."
- "Choose one."
- "Why is your name on this?"
- "You knew, didn't you?"

Do not overuse quoted lines. Use the pattern, not the exact wording.

### Add Public Consequence

If the action matters socially, reveal a ranking, rumor, article, live reaction, family summons, guild call, or institutional response.

### Add Rule Violation

If the world rule is important, end when the protagonist breaks it or learns the penalty.

## Commercial Repair Checklist

- Does the ending object appear on the page?
- Is the pressure caused by a previous action or decision?
- Can the reader guess possible outcomes?
- Is there a near-term payoff path?
- Does the hook match the genre reward?
- Did the episode give at least one reward before asking the reader to wait?

If the answer to reward and concrete object is no, rebuild the ending.
