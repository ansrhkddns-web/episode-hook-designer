# Genre Hook Matrix

Use this reference to adapt episode rewards, cliffhangers, comment triggers, and paid-conversion turns by genre. Do not apply a generic hook when genre readers expect a specific reward.

## Rule

For commercial episodes, every hook should speak the genre's purchase language:

- Hunter/tower readers buy growth, loot, rank, danger, and hidden rules.
- Regression/revenge readers buy prior knowledge, reversal, punishment, and variable control.
- Romance/rofan readers buy relationship distance, social pressure, status, jealousy, and public choice.
- Academy readers buy evaluation, rivalry, hierarchy, hidden talent, and public proof.
- Professional readers buy competence, case resolution, reputation, and a bigger client/problem.

## Genre Fit Check

Before finalizing a hook, ask:

- Does the reward match why readers clicked this genre?
- Does the ending object belong naturally to the genre?
- Does the comment trigger invite the expected reader behavior?
- Does the paid turn increase the same promise instead of switching genres?

If a genre has mixed promises, name the primary genre first and let the secondary genre provide flavor, not the main hook.
